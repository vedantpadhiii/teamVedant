🆘 QUICK TROUBLESHOOTING
================================================================================

"npm: command not found"
  → Install Node.js from nodejs.org

"npm run dev fails"
  → Delete node_modules, run npm install

"Port 3000 in use"
  → Close other processes or try: npm run dev -- --port 3001

"Styling looks broken"
  → Clear cache: Cmd+Shift+R or Ctrl+Shift+R

"Build fails"
  → Delete dist/, run npm run build again

For more: See README.md > Troubleshooting
